# Premier League Fantasy (FPL) Rules — 2025/26 Season

## Squad Selection

- **Budget**: £60 million to pick a squad of **9 players**
- **Formation**: 2 Goalkeepers, 3 Defenders, 3 Midfielder 1 Forwards or 2 Midfielder 2 Forward
- **Starting XI**: Pick 6 players each Gameweek (must include at least 1 GK, 2 DEF, 2or1 MID, 2or1 FWD)
- **Captain**: Earns double points; Vice-Captain steps in if Captain doesn't play
- **Max 2 players** from any Top Players List
- **Unlimited free transfers** before Gameweek 1 deadline
- **Captain** Person cant keep themselves captain 

## Scoring System

### Goals

| Position   | Points per Goal |
| ---------- | --------------- |
| Goalkeeper | 10              |
| Defender   | 6               |
| Midfielder | 5               |
| Forward    | 4               |

### Assists

- **3 points** per assist
- An assist is awarded to the player who makes the final pass before a goal


### Clean Sheets

| Position              | Points |
| --------------------- | ------ |
| GoalKeeper            | 10     |
| Defender              | 4      |
| Midfielder            | 1      |
| Forward               | 0      |

- Clean sheet points are awarded only if the player plays **30+ minutes**
- A clean sheet is voided if a goal is conceded while the player is on the pitch

### Saves (Goalkeepers)

- **1 point** for every **3 saves** made

### Penalty Saves

- **8 points** per penalty save

### Minutes Played

| Condition            | Points |
| -------------------- | ------ |
| Playing 35+ minutes  | 2      |
| Playing 1–34 minutes | 1      |


### Negative Points

| Action                                       | Points |
| -------------------------------------------- | ------ |
| Yellow card                                  | -1     |
| Red card                                     | -3     |
| Own goal                                     | -2     |
| Penalty miss                                 | -4     |
| Every 4 goals conceded (Goalkeeper/Defender) | -1     |

## Transfers

- **1 free transfer** per Gameweek
- Additional transfers cost **-4 points** each
- Player prices change based on transfer activity (supply & demand)

## Chips (2 Sets for 2025/26)

Each manager gets **two sets of chips** — 8 total. The first set must be used before Gameweek 19 (30 December 2025).

| Chip           | Effect                                                     |
| -------------- | ---------------------------------------------------------- |
| Triple Captain | Captain scores **triple** points instead of double         |
| Bench Boost    | All bench players score points for that Gameweek           |

- Only **one chip** can be played per Gameweek
- Chips cannot be cancelled once the Gameweek deadline passes


### Public Leagues

- Auto-joined at season start

## Deadlines

- Transfers and team changes must be made before the **Gameweek deadline** (Midnight before the first match kicks off)
- Captain and bench order can be changed until the deadline
